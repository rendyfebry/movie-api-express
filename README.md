# Movie DB API

Simple movie API using omdbapi database.

## Prerequisite

- Node v10.x
- NPM v6.x

## Installation

```
npm install
```

## Environment Variables

1. Copy .env file

```
cp .env.example .env
```

2. Fill the OMDB_KEY on .env file

## Running

Development mode

```
npm run dev
```

Production mode

```
npm start
```

## Test

```
npm run test
```
